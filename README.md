# ANM-SWITCH-IVENTARIO

Inventário de Ativos de Rede — Switches Gerenciáveis

Sede Brasília/DF e Unidades Regionais.

## Descrição

Aplicação web para levantamento e monitoramento de switches gerenciáveis da ANM (Agência Nacional de Mineração).

## Estrutura

- `src/lib/networkData.js` — Dados de inventário (SEDE_CORE, SEDE_ANDARES, REGIONAIS)
- `src/lib/generatePDF.js` — Geração de relatório PDF
- `src/pages/Report.jsx` — Dashboard principal
- `src/components/report/` — Componentes de tabelas e gráficos

## Tecnologias

- React 18
- Tailwind CSS
- jsPDF v4
- Recharts
- Framer Motion
- Lucide React
